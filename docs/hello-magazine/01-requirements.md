# 01. ハローマガジン 要件（スライド読み取り）

Hello! Magazine は「マレーシア教育移住・インターナショナルスクール」のリアルな
体験談・YouTube LIVE・Q&A・学校ランキングを扱う**コンテンツ（マガジン）面**。
Hello! プラットフォーム（学校検索・口コミ・写真ギャラリー等）の一部という位置づけ。

## マガジンTOP（スライド ⑲）の構成

```
ハローマガジンTOP
├ YouTube LIVE記事 … 卒ママ / 現ママ / 卒生徒
├ インタビュー記事 … 卒ママ / 現ママ / 卒生徒
├ よくある質問集   … 生活の知恵 / 入学前の準備
├ ランキング       … ★評価カテゴリー別ランキング
└ エージェント比較 … エージェント詳細ページ（001, 002, 003 …）
```

- TOPは「すべて／よくある質問／YouTube LIVE／体験談・インタビュー／学校ランキング／
  エージェント比較」でフィルタできる記事一覧を持つ。
- **タグを押すと、その種別（質問なら質問、YouTubeならYouTube）が表示される**ようにしたい。
- タグ体系は **CMSのタグ管理と同一**にそろえる（別スライド参照）。

## 記事種別ごとの要件

### 1. YouTube LIVE（MAG-xxxx / LIVE-xxx）
- 会員限定の YouTube LIVE アーカイブ記事。
- 目次（チャプター）、要約Q&A、LIVE概要（配信形式・出演者・テーマ・開催日）。
- **要望**: 「その秒数の画面をサムネイルみたいに出したい」「タップしたらその画面から再生」。

### 2. よくある質問（MAG-xxxx / FAQ-xxx ＋ まとめ）
- 「＋」ボタンで開閉する**アコーディオン型**Q&A。
- セクション分け（A.情報収集・学校選び / B.出願・入学プロセス / C.英語力・学習準備 /
  D.学校見学・雰囲気 等）。
- 「まとめ」記事は ②〜④ の区分がある：
  - ② 卒業生が答える：入学検討中の生徒向け
  - ③ 保護者が答える：在学中の保護者向け
  - ④ 卒業生が答える：在学中の生徒向け
  - （①保護者が答える：入学検討中の保護者向け を含め 2軸×2軸）

### 3. インタビュー（MAG-xxxx / INT-xxx）
- 卒業生・在校生・保護者の声。
- プロフィール（学校名・学区・立場・在学/卒業年）＋ カテゴリー別Q&A
  （授業・先生 / クラス・友人関係 / アクティビティ・行事 / 将来・進学）。

### 4. ランキング（MAG-xxxx / RANK-xxx）※一覧＋詳細あり
- 切り口：IB認定校 / STEM強化校 / 初年度費用が安い順 / 日本語サポートあり校 /
  エリアから探す / 口コミ評価から探す 等。
- 詳細：比較条件（対象エリア・費用基準・学年区分）＋ 学校エントリ
  （順位・学校名・★評価4項目＋総合・エリア・カリキュラム・初年度費用・特徴・写真）。
- 10件表示・ページネーション（100件登録があれば100まで）。
- **教育サイトなので王冠等のデザインはしない**。
- **要望**: フロント検索ページの検索結果をランキングに**自動挿入**できるか
  （食べログ風に学校カード自動表示）。予算的に無理なら手動でOK。
- ランキングは一覧バナーを自分で追加し、詳細ページで様々なランキングを作っていきたい。

### 5. エージェント比較 ※一覧＋詳細あり
- 教育移住エージェントの比較。
- タイプタグ（サービススタンス / 強み / ポジション=価格傾向）。
- 得意なサポート分野：項目ごとに ★（最大5）、**合計20まで**配分。未対応は「－」/グレーアウト。
- 体制タグ（日本語スタッフ在籍 / 現地法人あり / 日本法人あり / IB対応実績あり / 幼児校対応実績あり）。
- 得意な価格帯・カリキュラム・強い地域。
- 実績（直近3年の件数・対応校例）、スタンス、こんなご家庭におすすめ、費用、難題事例、
  会社基本情報（会社名・設立年・対応言語・所在地・連絡手段・レスポンス目安）、公式URL。
- 公式サイトへは**計測のため中間ページを挟んで**遷移。
- **将来**: 学校ページに口コミを紐づけ、口コミ＋返信（エージェントが回答）を
  Google口コミ風に。初期は猪口さん手動登録でOK。

## 運用方針
- マガジンは**すべてワイヤーに沿ったページ**を作成し、
  **猪口さんがコピペで様々な記事を展開**していけるようにする。
- 記事の上部は極力同じフォーマットにそろえる。
- 一覧＋詳細の2階層を持つのは **ランキング** と **エージェント比較** のみ。

## 共通UI（フッター等で繰り返し登場）
- 多言語切替：日本語 / English / 中文（简体） / 한국어 / Bahasa Melayu。
- 会員系導線：ログイン / 無料トライアル / 有料会員（機能比較ページ）/ 口コミ投稿。
- フッターリンク：利用規約 / プライバシーポリシー / 特定商取引法に基づく表記 / 投稿ガイドライン / お問い合わせ。
- Instagram 連携。
